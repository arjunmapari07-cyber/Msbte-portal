# MSBTE Portal

## 📌 Project Description
This is a simple MSBTE (Maharashtra State Board of Technical Education) portal project.
It is designed to provide students with easy access to study materials, information, and resources.

## 🚀 Features
- User-friendly interface
- Simple and clean design
- Easy navigation
- Static webpage using HTML

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript (if used)

## 📂 Project Structure
- index.html - Main webpage

## ▶️ How to Run
1. Download or clone the repository
2. Open `index.html` in any browser
